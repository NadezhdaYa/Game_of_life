# Conway's Game of Life (Console Edition)

Реализация игры «Жизнь» Д. Конвея в консольном режиме с возможностью сохранения истории поколений в текстовый файл и генерации PNG-снимков состояния поля. Проект разработан с соблюдением принципов модульности, строгой типизации и разделения ответственности.

## 🎯 Цель проекта
Выполнить Задание 5: реализовать симуляцию клеточного автомата, которая:
*   Читает начальную конфигурацию из текстового файла.
*   Выполняет заданное количество шагов симуляции.
*   Записывает историю всех поколений в выходной файл.
*   Генерирует серию PNG-изображений для каждого шага.
*   Использует оттенки базового цвета для отображения «возраста» живой клетки.

## 📁 Структура проекта

Проект разбит на модули для удобства тестирования и поддержки:

*   `main.py` — точка входа, обработка аргументов командной строки (CLI), оркестрация процесса.
*   `life_engine.py` — чистая бизнес-логика игры (правила Конвея, подсчет соседей, эволюция, учет возраста клеток). Не зависит от файловой системы и графики.
*   `file_io.py` — работа с текстовыми файлами (чтение начальной конфигурации, запись истории поколений).
*   `image_renderer.py` — генерация изображений с использованием библиотеки Pillow. Реализует логику интерполяции цвета в зависимости от возраста клетки.
